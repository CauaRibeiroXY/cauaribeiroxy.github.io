Tag <input/>

Especifica um campo de entrada onde o usuário pode inserir dados.

< input type="text" id="name" name="username"/>

Definição

  • Especifica um campo de entrada onde o usuário pode inserir dados.

    É o elemento de formulário mais importante.

    O elemento pode ser exibido de várias maneiras, dependendo do atributo type.

Atributos

Atributo Valor Descrição
accept file_extension audio/* video/* image/* media_type Especifica um filtro para quais tipos de arquivo o usuário pode escolher na caixa de diálogo de entrada de arquivo (apenas para type = "file")
alt text Especifica um texto alternativo para imagens (apenas para type = "image")
autocomplete on off Especifica se um elemento < input /> deve ter o preenchimento automático habilitado
autofocus  autofocus  Especifica que um elemento <input> deve receber foco automaticamente quando a página carrega
 checked  checked  Especifica que um elemento <input> deve ser pré-selecionado quando a página carrega (para type = "checkbox" ou type = "radio")
 dirname  inputname.dir  Especifica que a direção do texto será enviada
 disabled  disabled  Especifica que um elemento <input> deve ser desativado
 form form_id   Especifica a forma à qual o elemento <input> pertence
 formaction  URL  Especifica o URL do arquivo que irá processar o controle de entrada quando o formulário for enviado (para type = "submit" e type = "image")
 formenctype   application/x-www-form-urlencoded
multipart/form-data
text/plain		  Especifica como os dados do formulário devem ser codificados ao enviá-los ao servidor (para type = "submit" e type = "image")
 formmethod  get
post		  Define o método HTTP para enviar dados para o URL de ação (para type = "submit" e type = "image")
 formnovalidate  formnovalidate  Define que os elementos do formulário não devem ser validados quando enviados
 formtarget  _blank
_self
_parent
_top
framename		  Especifica onde exibir a resposta recebida após o envio do formulário (para type = "submit" e type = "image")
 height  pixels  Especifica a altura de um elemento <input> (apenas para type = "image")
 list  datalist_id  Refere-se a um elemento <datalist> que contém opções predefinidas para um elemento <input>
