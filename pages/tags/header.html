Tag <header/>

Um cabeçalho para um article:

<article>
<header>
<h1>A heading here</h1>
<p>Posted by John Doe</p>
<p>Some additional information here</p>
</header>
<p>Lorem Ipsum dolor set amet....</p>
</article>

A heading here

Posted by John Doe

Some additional information here

Definição

    • O elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

      Um elemento normalmente contém:
      ->um ou mais elementos de título(h1-h6)
      ->logotipo ou ícone
      ->informações de autoria

      Você pode ter vários elementos em um documento HTML. No entanto, não pode ser colocado dentro de um pédia, dress ou outro elemento.

Atributos

A <header> tag também suporta os atributos globais em HTML .

Exemplo

<header>
<h1>Main page heading here</h1>
<p>Posted by John Doe</p>
</header>
