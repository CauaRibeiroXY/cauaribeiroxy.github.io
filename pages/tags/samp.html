Tag <samp/>

Defina algum texto como saída de amostra de um programa de computador em um documento:

<p>Message from my computer:</p> <p><samp>File not found.<br>Press F1 to continue</samp></p>

Message from my computer:

File not found.
Press F1 to continue

Definição

    • A tag é usada para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespacial padrão do navegador.

Atributos

A <samp> tag também suporta os atributos globais em HTML .

Exemplo

<samp>计算机样本</samp><br>
