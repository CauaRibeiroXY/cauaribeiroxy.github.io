Tag template

Use <template> para manter algum conteúdo que ficará oculto quando a página for carregada.

<template>

<h2>Flower</h2>

<img src="img_white_flower.jpg" width="214" height="204">

</template>

Definição

  • A <template>tag é usada como um contêiner para manter algum conteúdo HTML oculto do usuário quando a página é carregada.

    O conteúdo interno <template>pode ser renderizado posteriormente com um JavaScript.

    Você pode usar a <template>tag se tiver algum código HTML que deseja usar repetidamente, mas não antes de solicitá-lo. Para fazer isso sem a <template>tag, você deve criar o código HTML com JavaScript para evitar que o navegador renderize o código.

Atributos

A tag <template> suporta todos os atributos globais em HTML .

Exemplo

<template>

<h2>Flower</h2>

<img src="img_white_flower.jpg" width="214" height="204">

</template>
