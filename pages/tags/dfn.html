Tag <dfn/>

Marque um termo com :

<dfn>HTML</dfn> is the standard markup language for creating web pages.

HTML is the standard markup language for creating web pages.

Definição

    • A tag significa o "elemento de definição", e especifica um termo que será definido dentro do conteúdo.

      O pai mais próximo da tag também deve conter a definição/explicação para o termo.

      O termo dentro da tag pode ser qualquer um dos seguintes:

Atributos

A <dfn> tag também suporta os atributos globais em HTML .

Exemplos

1. Assim como o conteúdo do elemento: dfn:

<dfn>HTML</dfn> is the standard markup language for creating web pages.

HTML is the standard markup language for creating web pages.

2. Ou, com o atributo do título adicionado: dfn:

<p><dfn title="HyperText Markup Language">HTML</dfn> is the standard markup language for creating web pages.</p>

HTML is the standard markup language for creating web pages.

3. Ou, com uma abbr ta dentro do elemento: dfn:

<p><dfn><abbr title="HyperText Markup Language">HTML</abbr></dfn> is the standard markup language for creating web pages.</p>

HTML is the standard markup language for creating web pages.

4. Ou, com o atributo id adicionado. Em seguida, sempre que um termo é usado, ele pode se referir à definição com uma a tag:

<p><dfn id="html-def">HTML</dfn> is the standard markup language for creating web pages.</p> <p>This is some text...</p> <p>This is some text...</p> <p>Learn <a href="#html-def">HTML</a> now.</p>

HTML is the standard markup language for creating web pages.
