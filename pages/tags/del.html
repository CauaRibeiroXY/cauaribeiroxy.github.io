Tag <del/>

Um texto com uma parte excluída e uma nova parte inserida:

<p>My favorite color is <del>blue</del> <ins>red</ins>!</p>

My favorite color is blue red!

Definição

    • A tag define texto que foi excluído de um documento. Os navegadores geralmente atacam uma linha através de texto suprimido.

Atributos

Atributo Valor Descrição
cite URL Especifica um URL para um documento que explica o motivo pelo qual o texto foi excluído / alterado
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e hora de quando o texto foi excluído / alterado

Exemplo

<html>
<head>
<style>
del {background-color: tomato;}
ins {background-color: yellow;}
</style>
</head>
<body>
<p>My favorite color is <del>blue</del> <ins>red</ins>!</p>
</body>
</html>


My favorite color is blue red!
