Tag source

Um reprodutor de áudio com dois arquivos de origem. O navegador escolherá o primeiro <source> que suporta:

<audio controls>

<source src="horse.ogg" type="audio/ogg">

<source src="horse.mp3" type="audio/mpeg">

Your browser does not support the audio element.

</audio>

Definição

  • O elemento source é utilizado para especificar múltiplos recursos de mídia de elementos <picture>, <audio> ou <video> em HTML5. É um elemento vazio.

Atributos

Atributo Valor Descrição
media media_query Aceita qualquer consulta de mídia válida que normalmente seria definida em um CSS
sizes Especifica tamanhos de imagem para diferentes layouts de página
src URL Requerido quando <source> é usado em <audio> e <video>. Especifica o URL do arquivo de mídia
srcset URL Requerido quando <source> é usado em <picture>. Especifica o URL da imagem para usar em diferentes situações
type MIME-type Especifica o tipo MIME-type do recurso

Exemplo

<audio controls>

<source src="horse.ogg" type="audio/ogg">

<source src="horse.mp3" type="audio/mpeg">

Your browser does not support the audio element.

</audio>
