Tag <select/>

O elemento <select/> é usado para criar uma lista suspensa.

<select name="cars" id="cars"/>

Definição

  • O <select/> elemento é mais frequentemente usado em um formulário, para coletar a entrada do usuário.

    O nameatributo é necessário para fazer referência aos dados do formulário após o formulário ser enviado (se você omitir o nameatributo, nenhum dado da lista suspensa será enviado).

    O id atributo é necessário para associar a lista suspensa a um rótulo.

    As tags <option/> dentro do <select/> elemento definem as opções disponíveis na lista suspensa.

Atributos

Atributo Valor Descrição
 autofocus  autofocus Especifica que a lista suspensa deve obter o foco automaticamente quando a página é carregada 
 disabled  disabled  Especifica que uma lista suspensa deve ser desativada
 form  form_id  Define a qual formulário a lista suspensa pertence
 multiple  multiple  Especifica que várias opções podem ser selecionadas de uma vez
 name  name  Define um nome para a lista suspensa
 required  required  Especifica que o usuário deve selecionar um valor antes de enviar o formulário
 size  number  Define o número de opções visíveis em uma lista suspensa

Exemplo

<label for="instrumento"> Escolha seu instrumento</label>

<select name="instrumento" id="instrumento">

<option value="trp">trompete</option>

<option value="tpa">trompa</option>

<option value="tbn">trombone</option>

<option value="tub">tuba</option>

</select>
