Tag <canvas/>

Desenhe um retângulo vermelho na mosca, e mostre-o dentro do canvas element:

<canvas id="myCanvas">
Your browser does not support the canvas tag.br
</canvas>
<script>
var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.fillStyle = "#FF0000";
ctx.fillRect(0, 0, 80, 80);
</script>

Your browser does not support the canvas tag.

Definição

    • A tag é usada para desenhar gráficos, em tempo real, via scripting (geralmente JavaScript).

      A tag é transparente, e é apenas um contêiner para gráficos, você deve usar um script para realmente desenhar os gráficos.

      Qualquer texto dentro do elemento será exibido em navegadores com JavaScript desativado e em navegadores que não suportam.

>

Atributos

Atributo Valor Descrição
height pixels Especifica a altura da tela. O valor padrão é 150
width pixels Especifica a largura da tela. O valor padrão é 300

Exemplo

<canvas id="myCanvas" width="200" height="100"
style="border:1px solid #000000;">
</canvas>
