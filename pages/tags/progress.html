Tag <progress/>

Mostrar uma barra de progresso:

<label for="file">Downloading do jogo:</label>

<progress id="file" value="32" max="100"> 32% </progress>


32%

Definição

  • A tag <progress/> representa o andamento da conclusão de uma tarefa

Atributos

Atributo Valor Descrição
 max number   Especifica quanto trabalho a tarefa requer no total. O valor padrão é 1
 value number   Especifica quanto da tarefa foi concluída

Exemplo

 <progress value="42" max="100">42 %</progress>

42 %
