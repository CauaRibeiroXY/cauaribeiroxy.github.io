Referencias Html

Lista de Referencias

  • <input/>

    Especifica um campo de entrada onde o usuário pode inserir dados.

  • <select/>

    Elemento usado para criar listas drop-down.

  • <u/>

    Representa algum texto não articulado e com estilo diferente do texto normal.

  • <details/>

    Especifica detalhes adicionais que o usuário pode abrir e fechar sob demanda.

  • <progress/>

    Representa o andamento da conclusão de uma tarefa.

  • <blockquote/>

    Representa uma longa citação.

  • <ins/>

    Representa uma longa citação.

  • <mark/>

    Representa uma longa citação.

  • <pre/>

    A tag define texto pré-formado.

  • <canvas/>

    A tag é usada para desenhar gráficos, em tempo real, via scripting (geralmente JavaScript).

  • <dt/>

    A tag define um termo/nome em uma lista de descrição.

  • <header/>

    O elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

  • <var/>

    é usada para definir uma variável na programação ou em uma expressão matemática.

  • <cite/>

    define o título de um trabalho criativo.

  • <samp/>

    é usada para definir a saída de amostra de um programa de computador.

  • <dfn/>

    significa "elemento de definição" e especifica um termo que será definido no conteúdo.

  • <del/>

    define o texto que foi excluído de um documento.

  • <hr/>

    define uma quebra temática em uma página HTML.

  • <time/>

    Define uma hora específica (ou data e hora).

  • <body/>

    Define o corpo do documento.

  • <source/>

    É usada para especificar vários recursos de mídia para elementos de mídia

  • <caption/>

    Define uma legenda da tabela

  • <thead/>

    É usada para agrupar o conteúdo do cabeçalho em uma tabela HTML.

  • <template/>

    É usada como um contêiner para manter algum conteúdo HTML oculto do usuário quando a página é carregada.
